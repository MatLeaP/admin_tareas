Acerca de este Proyecto

    

Esta página está diseñada para coordinar un equipo de trabajo, donde cada operador tiene asignadas distintas tareas. Cada operador puede acceder al listado de tareas pendientes y marcarlas como realizadas.

El usuario administrador es quien tiene los permisos para editar y borrar.

Podemos visualizar y filtrar el listado de Operadores, Categorías y Tareas en cada solapa correspondiente. La web es muy fácil de navegar, ademas de intuitiva y práctica. Los invito a comprobarlo ustedes mismos!!
